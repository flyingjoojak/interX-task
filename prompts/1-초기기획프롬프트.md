에이전트 기반 통합 채용 솔루션

## 인터엑스의 12가지 핵심가치
1. 선도적/정량 목표의식 (목표의식) : 조직 목표와 연계하여 시장 판도를 바꿀 수 있는 도전적인 목표(X차원)를 설정, 선행/후행 지표(마일스톤)를 수치화하여 관리.
2. 초효율적 시간관리 (시간관리) : AI 등 가용한 리소스 활용을 통한 자동화, 확보된 시간을 고부가가치 업무에 집중하여 마감기한 엄수.
3. 집요한 끈기 (끈기) : 실패를 통해 전략을 빠르게 바꿔 반복 실행하고, 어려운 과제나 불확실성(모호함) 속에서도 집요하게 해답 도출.
4. 가치중심적 문제해결 (문제해결) : 고객 및 시장에 대한 깊은 공감을 바탕으로 문제의 본질을 파악, 재발 방지를 위한 조직 차원의 구조적 문제 해결책 공유.
5. 근본적 비판 사고 (비판적 사고) : 수치/데이터를 근거로, 기존 경험에 대한 비판적 통찰을 통해 최적의 전략 설계 및 근본적인 대안을 도출.
6. 혁신 프로세스 가속화 (지속적 개선) : AI 및 신기술을 업무에 적극 도입하여 기존 프로세스를 혁신하고 개선안을 제시 및 전파.
7. 최고수준의 결과지향 (정성스러움) : 반복적인 실수 방지 및 높은 품질 지속 유지, 고객/파트너 대상의 모든 결과물에 최고 수준의 전문성 지향.
8. 자발적 성장동기 (자기동기부여) : 자신의 일의 의미와 가치를 명확히 인식, 강박적 호기심을 동력으로 자율적인 성장 궤도를 설정하며 업무 리드.
9. 미래낙관적 도전 (긍정적 태도) : 예상치 못한 변화(위기 상황)에도 '끝내주는 낙천성'을 유지, 더 나은 미래를 믿고 주변 사람의 사기에 긍정적 영향 제공.
10. 성장지향 피드백 (솔직한 피드백) : 피드백을 정기적으로 요청 및 제공, 성장지향적으로 상대방의 피드백 수용하고 감사함을 표현, 솔직한 소통 문화 구축.
11. 관계기반 전략소통 (네트워크 활용) : 내외부 이해관계자와 전략적 네트워크를 형성하여 시너지를 통한 새로운 기회 창출.
12. 강박적 호기심 (호기심) : 새로운 분야에도 지치지 않는 호기심으로 질문과 학습에 지속함, 실무에 선제적 적용.


## 1. 요구사항

### 데이터 추출 및 구조화 (주제 1):
지원자의 이력서/포트폴리오(PDF, 이미지) 업로드 및 OCR 파싱.
비정형 데이터에서 핵심 경험을 추출하여 구조화된 데이터(JSON/DB) 형태로 구조화.
추출된 경험을 인터엑스 12가지 핵심가치와 매칭하여 점수화.

### 이중 검증 면접 엔진 (주제 2):
이력서 데이터 기반의 사실관계 모순 탐지 로직(Cross-check).
의심 지점이나 심화 질문이 필요한 부분에 대한 '압박 면접 질문' 자동 생성.
질문에 대한 답변 중 과장되거나 AI가 생성한 것으로 의심되는 지점을 찾아내어, 논리적 허점을 파고드는 '압박 면접 꼬리 질문' 자동 생성. 
LangGraph 등을 활용한 에이전트 기반의 추론 프로세스 구축.

### 채용 의사결정 대시보드 (주제 3):
가치 매칭 결과 및 AI 사전 분석 보고서 및 요약 보고서를 시각화하여 면접관에게 웹 대시보드 상에 노출.
후보자별 분석 리포트를 대시보드 상에서 확인 및 PDF 출력 기능.
인터랙티브한 UI를 통해 면접관이 AI의 분석 근거를 확인할 수 있는 인터페이스.

## 2. 구현 디테일
언어 및 프레임워크: Python 기반 FastAPI (Backend) + react + next.js (Frontend).
AI 및 에이전트: * LangChain / LangGraph: 전체 분석 및 검증 워크플로우 제어.
모델 : 모델은 너가 추천해줘야함. API 기반 혹은 로컬 모델 기반 등.
데이터베이스 및 추출 기술은 너가 추천해줘야함.

## 3. 제약조건
인프라: 로컬에서 실행되는 웹 구축
보안: 보안과 데이터베이스 관련된 부분은 추천안을 제시해줘야함.
UI/UX: * 화면 구성과 UI 요소는 의사결정에 필요한 정보만 담아 최소화.
디자인: 인터엑스의 전문성을 반영하여 포인트 색상 1가지를 제외하고 모두 무채색(White, Gray, Black) 사용.
복잡도: 최대한 에러가 적고 구현이 명확한 라이브러리 위주로 선정하여 안정성 확보.

## 4. 완료 기준
파이프라인 연동: PDF 업로드 시 데이터 추출 -> 검증 질문 생성 -> 대시보드 시각화까지의 전 과정이 단절 없이 작동함.
분류 정확도: 인터엑스의 12가지 가치 카테고리에 맞춰 지원자의 경험이 논리적으로 분류됨.
질문 생성 능력: 이력서 내용과 모순되거나 보완이 필요한 지점을 정확히 짚어내는 고차원적인 꼬리 질문 리포트가 생성됨.
질문 생성 능력2 : 질문에 대한 후보자의 답변에서 과장된 점이나 AI가 생성한 것으로 의심되는 지점을 찾아내 논리적 허점을 파고드는 압박 꼬리 질문이 생성됨. 
시각화 완성도: 면접관이 후보자의 강점과 약점을 한눈에 파악할 수 있는 AX(AI Experience) 기반 대시보드가 구동됨.
결과물 제출: 실제 동작하는 소스코드, 구동 영상, 그리고 AI와 협업한 과정이 담긴 기술 기술서 포함.

# PRD — InterX 에이전트 기반 통합 채용 솔루션

## 제품 목적
면접관이 지원자 이력서/포트폴리오를 업로드하면, AI가 인터엑스 12가지 핵심가치 기준으로 분석하고 압박 꼬리질문을 생성한다. 서류 심사와 실시간 면접 보조를 단일 플랫폼에서 처리한다.

## 사용자
인터엑스 면접관. 로컬 환경 단일 머신, 복수 면접관이 동일 후보자 열람 가능. JWT 로그인 필수.

## 핵심가치 (스코어링 기준)
| # | 가치명 | 키워드 |
|---|--------|--------|
| 1 | 목표의식 | 선도적/정량 목표, 마일스톤 수치화 |
| 2 | 시간관리 | AI 활용 자동화, 마감기한 엄수 |
| 3 | 끈기 | 실패 후 재실행, 모호함 속 해답 도출 |
| 4 | 문제해결 | 고객 공감, 구조적 재발 방지 |
| 5 | 비판적사고 | 데이터 기반, 기존 경험 비판적 통찰 |
| 6 | 지속적개선 | AI/신기술 도입, 프로세스 혁신 |
| 7 | 정성스러움 | 반복 실수 방지, 최고 수준 결과물 |
| 8 | 자기동기부여 | 업무 의미 인식, 자율적 성장 |
| 9 | 긍정적태도 | 위기 속 낙천성, 주변 사기 영향 |

| 12 | 호기심 | 새 분야 학습 지속, 선제적 실무 적용 |

## 기능 요구사항

### F1. 데이터 추출 및 구조화
- 이력서/포트폴리오 업로드 (이력서: PDF/JPG/PNG/PPT/PPTX, 포트폴리오: PDF/JPG/PNG)
- OCR 파이프라인으로 텍스트 추출 → 구조화 JSON (경력/학력/기술/성과)
- 12가지 핵심가치 매핑 + 점수화 (0~100점, 동등 가중치)
- 문서 신뢰도 점수 산출 (일치도+신뢰도+완성도, 0~100점)

### F2. 이중 검증 면접 엔진
- 이력서 내 모순 탐지 (날짜/직책/기간/성과 Cross-check)
- 이력서↔포트폴리오 충돌 → 우선순위 없이 허점 기록 + 압박질문 생성
- 사전 압박 질문 자동 생성 (LangGraph 에이전트)
- 면접 중 답변 입력 → 실시간 압박 꼬리질문 생성 (목표: 10초 이내)
- AI 생성 답변 탐지 대신 Drill-down 질문으로 검증 (탐지 오탐 리스크 회피)

### F3. 채용 의사결정 대시보드
- 12가지 가치 레이더 차트 + 가치별 점수 클릭 시 AI 근거 패널
- 홈 화면에서 후보자 간 점수 비교 (별도 화면 없음)
- 사전 분석 리포트 + 면접 세션 통합 PDF 출력 (Playwright)
- 후보자 상태 9단계 관리 (자동 3개 + 수동 6개)

## 비기능 요구사항
- **안정성 우선**: 에러가 적고 구현이 명확한 라이브러리 선택
- **정확도 우선**: 안정성과 정확도 트레이드오프 시 정확도 선택
- **로컬 실행**: 외부 배포 없음, Docker 불필요
- **한국어 전용**: OCR, 프롬프트, UI 모두 한국어 기준
- **보안**: PII 익명화 후 API 전송, 로그 마스킹, JWT 인증

## 완료 기준
1. PDF 업로드 → 분석 → 대시보드까지 단절 없이 작동
2. 12가지 가치 분류가 이력서 내용에 논리적으로 연결됨
3. 이력서 모순/허점을 정확히 짚는 압박 꼬리질문 생성
4. 답변의 모호함/과장을 파고드는 실시간 꼬리질문 생성
5. 면접관이 후보자 강점/약점을 한눈에 파악 가능

## 제약
- 인프라: 로컬 실행, 배포 없음
- 포트폴리오 PPT/PPTX 미지원 (LibreOffice 의존성 회피)
- 80점 기준선은 참고용, 합불 결정에 직접 개입하지 않음
- UI: 포인트 컬러 #ff8000 단색, 나머지 무채색
